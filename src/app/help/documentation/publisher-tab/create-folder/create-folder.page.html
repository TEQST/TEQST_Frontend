Creating a folder

If the admin has promoted you to a publisher a new publisher icon will be shown in the tab bar  [1].
By clicking on it, you will be redirected to an overview of all folders you have created (initially, the list will be empty).
The plus button  [2] allows you to create new folders and texts.
In a folder can exist whether subfolders or texts.
You can delete a folder by pressing the trashcan button  [3] in the same row.
